# 📧 Spam Mail Prediction

## 📌 Overview
This project is a Machine Learning-based system that classifies emails as spam or not spam.

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLP (TF-IDF)

## 📊 How it Works
1. Data preprocessing (cleaning text)
2. Feature extraction using TF-IDF
3. Model training using classification algorithms
4. Prediction of spam or non-spam messages

## ▶️ How to Run
1. Open the notebook in Google Colab
2. Run all cells step by step
3. Enter your own message to test prediction

## 📈 Output
- Spam ❌
- Not Spam ✅
